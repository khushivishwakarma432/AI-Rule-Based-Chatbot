# AI Rule-Based Chatbot 🤖

This is a simple AI chatbot built using Python.

## Features
- Rule-based chatbot
- Random responses
- Username personalization
- System time display
- Data handling using Pandas

## Technologies Used
- Python
- Pandas
- Random Library

## How to Run

1. Install Python
2. Install pandas
3. Run the file

python chatbot.py
